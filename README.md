# Instruções

 - Clone o repositório
 - `$ cd vue-laravel`
 - `$ composer install`
 - Edite o arquivo `.env` com as informações de conexão com o BD
 - Execute o comando `php artisan key:generate`
 - Execute o comando `npm install`
 - Execute o comando `php artisan migrate --seed`
